# Unknown Trivia Application
St. Norbert College Computer Science Capstone project 2019

For my senior year of Computer Science. This is the creation of a Trivia App.

I have choosen to use Angular, RxJS, Jasmine testing framework, Ionic 4 Framework (aka so also cordova).

# How To Run:

Install Node.js and Ionic
  Instructions can be found on there website

go to root folder

run ionic serve -c 
  this will bring up running instance of the application
